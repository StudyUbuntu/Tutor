#### 2. 学习使用git下载上传文件，随意尝试。
> 简易指导

* ubuntu下下载安装

  `
   sudo apt-get install git  
  `
* 使用git下载文档

  `
  git clone https://github.com/UbuntuLeaner/UL.git
  `
* 使用文本编辑器修改或拷贝文档到目录
* 注册github帐号，并加入StudyUbuntu组织
* 提交修改

  `
  git commit -a -m "Update"
  `

  `
  git push origin master
  `
